## 3. 发布物料(社群 · 英文定稿,可直接发)

### Show HN(news.ycombinator.com → Submit)
**标题(≤80 字符)**
```
Show HN: DockDocs – 50+ PDF tools in your browser + AI that cites the source
```
**正文**
```
Hi HN — I built DockDocs (https://dockdocs.app), a privacy-first PDF toolkit.

Most online PDF tools upload your file to a server. DockDocs runs the common ones (compress, merge, split, convert, OCR, watermark, sign, redact) entirely in your browser with pdf-lib / pdf.js — your file never leaves your device. ~50 tools, free, no account.

On top of that there's an AI layer that actually reads documents: chat with a PDF where every answer links back to the source text, summarize long reports, compare multiple documents side by side, and extract invoice/table data to a spreadsheet.

It's free to start; AI + larger files + batch are paid. Would love feedback on the tools and the "cite the source" AI approach.
```
**时机**:美东周二–周四 9:00–11:00。发后**守在评论区**回复(前 2 小时最关键)。

## 4. 执行顺序(给 Joe 的 checklist)
1. **先 Product Hunt 选好发布日**(周二–周四),其他目录可随时铺。
2. **P0 八个目录全提交**(1 小时左右)。
3. 发 **Show HN**(挑好时机)+ **2-3 个 subreddit**。
4. 之后每天铺 **5-8 个 P1/P2 目录**,一两周铺完。
5. 截图/Logo 统一用 https://dockdocs.app/opengraph-image。

## 5. 怎么看是否起效(2-4 周后)
- **GSC →「链接数量」→ 外部链接**:从 0 开始往上涨 = 外链在进来。
- **GSC →「网页索引编制」**:"已发现未收录 663" 应开始下降、"已编入索引 215" 上涨。
- **GSC →「效果」**:平均排名(现 65)往前走、点击数从个位数往上。
- 这些是滞后指标,**外链进来后通常 2-6 周**才反映到排名。
